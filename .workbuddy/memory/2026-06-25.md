# 2026-06-25

## 家庭空间加入功能
- 后端 `services/domain.go`:
  - `FamilyService.Join(code, userID, username)`: 验证邀请码 → 检查是否已成员 → 创建 FamilyMember 记录 → 返回 FamilySpace
  - `FamilyService.ListInvites(spaceID)`: 列出有效邀请
- 后端 `handlers/domain.go`:
  - `POST /api/v1/family/:id/join` — 通过邀请码加入家庭
  - `GET /api/v1/family/:id/invites` — 查看邀请列表
- 前端 `ProfilePage.tsx`:
  - "加入家庭" 按钮 → 输入邀请码 → 调用 `joinFamily` API
  - 家庭成员列表展示姓名+权限标签
- 权限体系扩展: `edit` (可编辑) / `upload` (可上传) / `view` (仅查看)

## 邀请码自定义身份
- `InviteModal`: 预设身份按钮扩展为 8 个 (妈妈/爸爸/爷爷/奶奶/外公/外婆/哥哥/姐姐)
- 新增自定义身份输入框 (如 "叔叔" "姑姑")
- 默认权限改为 `upload`

## 修改密码功能
- 后端: `AuthHandler.ChangePassword` → `PUT /api/v1/profile/password`
- `auth_service.ChangePassword`: 验证旧密码 → bcrypt 对比 → 新密码 bcrypt hash → 更新
- 前端 ProfilePage: "修改密码" 入口 → 旧密码/新密码/确认密码 表单

## 事件相册编辑 & 删除
- 后端: `AlbumService.Update(id, title, desc)` + `PATCH /api/v1/albums/:id`
- 前端 `AlbumPage`:
  - 相册卡片长按 500ms → 弹出操作浮层 (编辑/删除)
  - 编辑: 修改标题+描述，调用 `updateAlbum`
  - 删除: confirm → 调用 `deleteAlbum` → 回退列表
- `store`: 新增 `updateAlbum`、`deleteAlbum` 方法

## 创意作品视频支持
- `CreativeWorkForm`: 新增视频上传 (type=video 时显示视频选择器)
  - `videoFile` + `videoPreview` state，`handleVideoUpload`/`removeVideo`

- `CreativeWorkCard`: 视频型作品显示 `<video>` 播放器 (16:9, 带控制条)
- `store`: `convertCreativeWork` 新增 `videoUrl` 字段映射
- `types/index.ts`: `CreativeWork` 接口新增 `videoUrl?: string`

## 首页权限控制
- `HomePage ActionBar`: 删除按钮受权限控制

  - 仅 `permission === 'edit'` 或 `isOwner` 时显示删除按钮
  - 未加入家庭空间时默认不显示
