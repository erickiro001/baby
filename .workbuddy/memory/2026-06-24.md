# 2026-06-24

## 搭建后端框架
- 技术栈: Go 1.26 + Gin + GORM + glebarez/sqlite (纯Go, 无CGO) + golang-jwt + bcrypt + Air 热更新
- 项目结构: `cmd/server/main.go` 入口, `internal/` 下按 config / database / models / services / handlers / middleware / utils 分层
- User 模型: ID, Username, Email, PasswordHash, CreatedAt, UpdatedAt, DeletedAt (软删除), PasswordHash 通过 `json:"-"` 禁止序列化返回
- 认证: 注册(用户名+邮箱唯一性校验, bcrypt cost=12), 登录(支持用户名或邮箱), JWT Bearer Token
- 安全中间件: JWT Auth (Bearer Token解析), CORS, SecurityHeaders (XSS/CSP/Frame), NoCache, SizeLimit (1MB), RateLimiter (Token Bucket, 10req/min per IP)
- API 路由: `/api/health`, `/api/v1/auth/register`, `/api/v1/auth/login` (公开+限流), `/api/v1/profile` (需JWT)
- 统一响应格式: {code, message, data}
- 配置: 通过 .env 文件 + 环境变量, 默认端口 8080, JWT 过期 24h
- Air 热更新已配置, Makefile 提供 dev/build/run/clean 命令
- 编译验证通过, 生成 35MB 可执行文件
- 服务已启动运行在 localhost:8080

## 中间件完备性审查
- 已有: Logger, Recovery, CORS(*), SecurityHeaders, NoCache, SizeLimit(1MB), RateLimiter(10/min/IP), JWTAuth
- 缺失待补: Request Timeout（防慢连接攻击）、Request ID（日志串联）、HSTS/CSP（生产安全头）
